## Proyectos de Alto Impacto (Featured Projects)

### 📊 [Enterprise Market Intelligence Platform | Big Data & RAG Architecture]
* **Enfoque:** Procesamiento Distribuido, Delta Lakehouse, Semantic Chunking, Bases Vectoriales, Mitigación de Alucinaciones.
* **Descripción:** Diseño e implementación de un pipeline end-to-end escalable para la ingesta, normalización y análisis semántico automatizado de la "Voz del Cliente" (Voice of Customer - VoC) sobre un corpus masivo de **1.68 millones de reseñas**. El sistema integra una capa distribuida en PySpark, una base de datos semántica para recuperación hiperdimensional de baja latencia y un orquestador generativo asimétrico.
* **Hitos de Ingeniería:**
  * Reducción del **44.9% en espacio de almacenamiento en disco** mediante la transición de esquemas JSON semiestructurados crudos a formatos binarios de persistencia inmutable Parquet.
  * Implementación de ingeniería de datos estricta llevando el set a la **Primera Forma Normal (1FN)** mediante desanidamiento vectorial de estructuras complejas nativas de Spark, eliminando funciones tradicionales .apply() o lambdas para proteger el paralelismo.
  * Segmentación textual avanzada mediante un divisor recursivo parametrizado (`chunk_size=150` palabras, `chunk_overlap=30` palabras) manteniendo trazabilidad absoluta por ID de registro para suprimir el fenómeno de pérdida de contexto (*"Lost in the Middle"*).
  * Orquestación RAG robusta con control estricto de temperatura, obligando al LLM a ejecutar directivas de abstención analítica ante contextos incompletos, erradicando alucinaciones informativas.

### 🧪 [Enterprise Management & Analytics Cloud Architecture (Ixea Eros)]
* **Enfoque:** Arquitectura de Software Enterprise, Control de Accesos Jerárquicos (RBAC) y Motores Analíticos en Tiempo Real.
* **Descripción:** Diseño estructural y desarrollo modular de una plataforma SaaS con **+100k líneas de código propio** para la automatización total de flujos operativos, comerciales y contables del sector retail. Optimización avanzada de queries e infraestructura en la nube sin dependencia de servidores locales, gestionando de forma concurrente el estado transaccional del negocio.

### 🤖 [High-Fidelity Customer Churn Prediction Pipeline | Interconnect]
* **Enfoque:** Gradient Boosting, MLOps, Redes Neuronales Artificiales (ANN), Mitigación de Riesgos Financieros.
* **Descripción:** Pipeline predictivo de punta a punta enfocado en detectar la deserción temprana de clientes para grandes empresas de telecomunicaciones. Implementación avanzada con **LightGBM** y **TensorFlow/Keras**, controlando estrictamente el *Data Leakage* mediante técnicas de validación cruzada y optimizando hiperparámetros en espacios multidimensionales.
* **Métricas Logradas:** **AUC-ROC de 0.9073** y una **Exactitud (Accuracy) global del 83.04%** sobre conjuntos de validación inéditos.
